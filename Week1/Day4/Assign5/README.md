# Truth Tables For Logical Operators

## AND Operator
| Boolean | Boolean | Result |
| ------- | ------- | ------ |
| True    | True    | True   |
| True    | False   | False  |
| False   | True    | False  |
| False   | False   | False  |

## OR Operator
| Boolean | Boolean | Result |
| ------- | ------- | ------ |
| True    | True    | True   |
| True    | False   | True   |
| False   | True    | True   |
| False   | False   | False  |

## NOT Operator
| Boolean | Result |
| ------- | ------ |
| True    | False  |
| False   | True   |
